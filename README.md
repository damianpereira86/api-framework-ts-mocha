# Installation
* On the root folder
* `npm install` 

# Setup your local env 
  * Copy the example.env
  * Rename it as .env
  * Update the following properties
    * BASEURL=
    * USER=
    * PASSWORD=

# Run tests
  ` npm test`  
  ` npm run smoke`  
  ` npm run regression`  

# How to use prettier - formatting tool
  * Check for issues: `npm run check` 
  * Resolve issues: `npm run prettify` 

# How to use eslint 
  ` npm run lint`
